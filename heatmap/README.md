## Scripts
### Develop
#### Start app in develop mode
```
npm run dev
```
#### Lint
```
npm run lint
```

### Production
#### Build app for production
```
npm run build
```
#### Start app in production mode
```
npm run start
```
#### Or build & start at the same time
```
npm run prod
```

### Database
#### Reset database
```
npm run db:reset
```
